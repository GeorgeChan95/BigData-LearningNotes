### man

#### 基本语法

man [命令或配置文件]    （功能描述：获得帮助信息）



#### 显示说明

| 信息        | 功能                     |
| ----------- | ------------------------ |
| NAME        | 命令的名称和单行描述     |
| SYNOPSIS    | 怎样使用命令             |
| DESCRIPTION | 命令功能的深入讨论       |
| EXAMPLES    | 怎样使用命令的例子       |
| SEE ALSO    | 相关主题（通常是手册页） |



#### 案例实操



##### 查看ls命令的帮助信息

```shell
man ls
```
